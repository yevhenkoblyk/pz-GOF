# Creational Design Patterns (TypeScript)

## Виконані патерни:

1. Singleton — один екземпляр класу  
2. Factory Method — створення обʼєктів через фабрику  
3. Abstract Factory — створення сімейства обʼєктів  
4. Builder — покрокове створення обʼєкта  
5. Prototype — клонування обʼєктів  

## Запуск прикладів

```bash
npx tsx src/creational/singleton/index.ts
npx tsx src/creational/factory/index.ts
npx tsx src/creational/abstract-factory/index.ts
npx tsx src/creational/builder/index.ts
npx tsx src/creational/prototype/index.ts
